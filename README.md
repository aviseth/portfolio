# Aviseth's Portfolio & Resume Website

A beautifully crafted portfolio website with an **anti-minimalist design** inspired by Art Deco, Victorian, Gothic, and Tudor architectural styles. This site combines elegant aesthetics with modern functionality, featuring an integrated AI assistant for answering questions about Aviseth's background.

## 🎨 Design Philosophy

This portfolio embraces **anti-minimalism** - the opposite of brutalist simplicity. Drawing inspiration from:
- **Art Deco** geometric patterns and rich color palettes
- **Victorian** ornate decorative elements
- **Gothic** architectural grandeur
- **Tudor** classical sophistication

## ✨ Features

### 🏛️ Rich Visual Design
- Ornate headers with gold gradients and decorative elements
- Art Deco-inspired color scheme (gold, deep blue, burgundy, cream)
- Victorian typography using elegant serif fonts
- Elaborate borders, frames, and ornamental details
- Smooth animations and hover effects

### 🤖 AI-Powered Chat Assistant
- **Intelligent Q&A**: Ask questions about Aviseth's experience, skills, education, and projects
- **Context-aware responses**: Understands different question types and provides detailed answers
- **Professional tone**: Perfect for recruiters and potential employers
- **Real-time interaction**: Instant responses based on resume data

### 📱 Modern Functionality
- **Responsive design** that works on all devices
- **Smooth scrolling navigation** with active section highlighting
- **Interactive elements** with hover effects and animations
- **Accessibility features** with proper semantic HTML
- **Performance optimized** with efficient CSS and JavaScript

## 🛠️ Technical Stack

- **HTML5** - Semantic structure
- **CSS3** - Advanced styling with gradients, animations, and responsive design
- **Vanilla JavaScript** - Interactive functionality and animations
- **Local AI** - Custom chat system with intelligent responses

## 🚀 Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/aviseth/portfolio.git
   cd portfolio
   ```

2. **Serve the website**
   ```bash
   # Using Python
   python3 -m http.server 8000
   
   # Using Node.js
   npx serve .
   
   # Or any other static file server
   ```

3. **Open in browser**
   Navigate to `http://localhost:8000`
